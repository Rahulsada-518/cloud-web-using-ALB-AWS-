# cloud-web-using-ALB-AWS-
# Scalable Web App with AWS ALB and Auto Scaling

This project demonstrates a scalable web application deployed on Amazon EC2 using Application Load Balancer and Auto Scaling Group.

## Architecture Flow

User → Application Load Balancer → Target Group → Auto Scaling Group → EC2 Web Servers

## AWS Services Used

- Amazon EC2
- Application Load Balancer
- Auto Scaling Group
- Launch Template
- Target Group
- Security Groups
